# こころの日記

認知行動療法（CBT）を活用したこころの回復をサポートする個人用日記アプリ。

## 構成

- `app.py` — Streamlit アプリ本体
- `requirements.txt` — 依存パッケージ
- `.env` — API キー（gitignore 対象）

## 機能

### Aモード（コラム法）
- 特定の出来事があったときに使用
- デフォルト5コラム（状況・自動思考・感情・認知の歪み・代替思考）
- Claude が状況を分析して7コラムを推薦する場合あり

### Bモード（対話）
- 特定の出来事がなかった日に使用
- Claude が最初の問いかけを行い、温かく対話する

### まとめ生成
- セッション終了後に「今日の気づき」を自動生成
- テキストで保存可能
- note への自動投稿は Phase 2 で対応予定

## 起動方法

```
起動.bat をダブルクリック
```

または

```
cd こころの日記
streamlit run app.py
```
